# 时光记账 #
## 用于练手项目，使用MVVM架构， ##
MVVM：  
Model：数据模型层。包含业务逻辑和校验逻辑。

View：屏幕上显示的UI界面（layout、views）。

ViewModel：View和Model之间的链接桥梁，处理视图逻辑。  

功能列表：
- 主要功能：记录账单；
- 可供用户登录；
- 可注册新用户；
- 数据的保存；
- 可查看历史记录；
- 生成统计图；
- 分类，分为周，月，年查看；
- 每日支出的预算；
- 密码修改；
- 统计总数的功能；
- 可切换主题；
- 软件说明；
